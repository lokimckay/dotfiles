# Dotfiles

My Windows + WSL environment dotfiles.

## Setup

1. Install apps:
   - [VS Code](https://code.visualstudio.com/)
   - [VS Code WSL extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)
   - [WSL](https://learn.microsoft.com/en-us/windows/wsl/install)
   - [Rust](https://rust-lang.org/tools/install/)
   - [Powertoys](https://learn.microsoft.com/en-us/windows/powertoys/install)

1. Run `dotboot`
1. Install packages: `. ~/.apt_packages`
1. Install VS Code extensions: `. $VSCODE_USER_HOME/.extensions.sh`

1. put home stuff there
1. source .bash_env
1. put all other stuff there
